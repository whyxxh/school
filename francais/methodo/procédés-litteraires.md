
# IV. Liste des procédés

## 1) Figures de style

* Comparaison : Avec un mot de comparaison (comme, tel, etc.)
  Exemple : Il s’envola, tel Superman.

* Métaphore : Comparaison sans mot de comparaison
  Exemple : Il est un vrai chinois.

* Métaphore filée : Métaphore prolongée dans le texte

* Métonymie : Désigner quelque chose par un élément qui y est lié
  Exemple : Paris se réveille (les habitants de Paris).

* Synecdoque : Type de métonymie : partie pour le tout
  Exemple : Une fine lame (un escrimeur).

* Allégorie : Idée abstraite sous forme humaine (majuscule)
  Exemple : La Mort arriva.

* Personnification : Donner des traits humains à une chose
  Exemple : Les ponts sur ma route me parlent.

* Réification : Inverse de la personnification
  Exemple : L’homme est devenu une machine.

* Prosopopée : Faire parler un objet, un mort, un absent
  Exemple : On est des tubes, on est pas des pots...

* Allitération : Répétition de consonnes
  Exemple : Serpents à sonnettes sifflent.

* Assonance : Répétition de voyelles
  Exemple : Menteur tout à l’heure t’as pas pris le beurre.

* Asyndète : Suppression de conjonctions
  Exemple : J’ai oublié les clés, la voisine en a un double.

* Polysyndète : Surabondance de conjonctions
  Exemple : Et j’ai des poireaux et des cailloux et...

* Phénomène d’écho / Répétition :
  Exemple : Elle est belle, ma voiture ? Ma voiture ?

* Pléonasme : Redondance inutile
  Exemple : Monter en haut.

* Anaphore : Répétition en début de phrase
  Exemple : Patate est belle. Patate est forte. Patate vaincra.

* Énumération : Liste séparée par virgules
  Exemple : J’aime les carottes, les bananes, le chocolat...

* Gradation : Progression croissante ou décroissante
  Exemple : Grande, immense, gigantesque !

* Chiasme : Structure croisée (ABBA)
  Exemple : Il faut manger pour vivre et non vivre pour manger.

* Hyperbole : Exagération
  Exemple : Je meurs de faim.

* Litote / Euphémisme : Dire moins pour suggérer plus
  Exemple : Il n’est pas très malin.

* Oxymore : Deux mots opposés côte à côte
  Exemple : Une obscure clarté.

* Parallélisme de construction : Répétition de structure
  Exemple : Les licornes mangent des poireaux / les chiens des croquettes.

* Périphrase : Remplacer un mot par une expression
  Exemple : La ville lumière (Paris).

* Prétérition : Dire qu’on ne va pas dire quelque chose, puis le dire
  Exemple : Je ne parlerai pas de Talia...

* Antiphrase : Ironie par inversion de sens
  Exemple : Bravo, tu as encore tout cassé !

* Antithèse : Opposition de deux idées
  Exemple : J’ai plus d’amour que tu n’as d’oubli.

* Paronomase : Paronymes proches phonétiquement
  Exemple : Il pleure alors qu’il pleut.

* Polyptote : Mots issus du même radical
  Exemple : Barbe, barbelé.

* Prolepse : Anticipation, parler du futur au présent

* Analepse : Retour en arrière (flashback)

* Hypotypose : Description très vivante d’une scène

Autres figures mentionnées (sans détails) :
Épanadiplose, expolition, accumulation, antimétabole, antapodose, gémination, suffixation, homéotéleute, homéoptote, polyptote, isolexisme

## 2) Autres procédés

### Texte en général

* Choix du vocabulaire : soutenu, courant, familier, argotique...

* Champs lexicaux (éviter d’en mettre plus de deux)

* Topos / topoï : Thèmes récurrents (amour, guerre…)

* Tonalités

* Rupture syntaxique
  Exemple : Je \_\_\_. Je \_\_\_. Nous \_\_\_.

* Phrases longues / adjectifs nombreux : volonté de détailler

* Ponctuation : influence le ton et le rythme

* Inégalité de parole : montre un rapport de pouvoir

---

### Temps du verbe

Il y a toujours quelque chose à dire sur le temps.

* Voie passive : il subit
  Exemple : Il était attaché.

* Participe passé : parfois adjectival
  Exemple : qu’il s’était imposé.

* Passé : temps révolu

* Passé composé : impact fort sur le présent

* Plus-que-parfait : retour plus ancien, parfois emphatique

* Présent de vérité générale : vérité universelle

* Impératif : donne des ordres → volonté forte

* Indicatif / futur : mode de la certitude

* Certains temps donnent une sensation de réel ou d’irréel

---

### Types de phrases

* Phrase nominale : sans verbe
  Exemple : Entre ces deux portes.

* Groupe nominal
  Exemple : Lucas, son chien et un mouton mangent.

* Phrase emphatique : ordre inversé pour insistance
  Exemple : Et partout il va me charmant.

* Tournure impersonnelle
  Exemple : Il pleut.

* Phrase négative

* Négation restrictive
  Exemple : Tu n’es qu’un...

* Phrase lapidaire : très courte

* Interjection
  Exemple : Ah !

* Citation : argument d’autorité
Exemple : “C’est la base”, comme dirait un sage alsacien.

* Question rhétorique : n’attend pas de réponse

* Didascalie : indication de mise en scène

---

### Mise en valeur de mots

* Apposition : mot ou groupe entre virgules
  Exemple : Allons de mes poireaux, hélas, trop petits.

* Antéposition de l’adjectif
  Exemple : secret malheur.

* Postposition (normale)
  Exemple : faiblesse extrême.

* Jeux de mots : homophonie, polysémie, etc.

---

### Adjectifs, noms

* Adjectif qualificatif

* Adjectif affectif
  Exemple : ma chère.

* Adjectif péjoratif / mélioratif

* Adjectif substantivé
  Exemple : le misérable.

* Substantif : synonyme chic de « nom »

* Néologisme : mot inventé
  Exemple : phébardage.

### Adverbes, compléments circonstanciels

* Complément circonstanciel : temps, lieu, manière, etc.

* Complément circonstanciel de temps itératif
  Exemple : sans cesse.

* Adverbe d’intensité
  Exemples : trop, si grave.

* Modalisateurs : marquent la subjectivité
  Exemples : il semble, peut-être, à peu près.

### Comparaison

* Comparatif : de supériorité / infériorité
  Exemples : plus, moins.

* Superlatif
  Exemples : le plus grand, le pire, la meilleure.

---

### Autres procédés

* Rythme ternaire : 3 éléments pour stabilité
  Exemple : vertu, gloire et beauté.

* Pronoms personnels : je, tu, il...

* Technique du blason : décrire et embellir le corps (souvent en poésie)
  Exemple : Tes cheveux sont soyeux, tes yeux reflètent l’océan…

* Verbes d’action

* Présentatifs
  Exemples : c’est, voici, voilà.

* Conjonctions de subordination
  Exemples : puisque, lorsque, bien que…

* Sens factitif : faire + verbe
  Exemple : Il a fait manger de l’herbe au mouton.
