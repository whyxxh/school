# 1. une description de l'appart
- description idéalisée
    - longue phrases : desctiption detaillée voire lourde
    - champ lexical de bois, du mobilier : on est sur une description tres réaliste et detaillée presque comme un plan, venant de
    - utilisation de l'indicatif imparfait de description ajoute un espoir dans l'esprit du lecteur, un sens d'idéal est
    conservé l'imaginaire mais basé dans la realité
    - "un meme ... un meme ... un meme ..." on a une enumeration anaphorique du bureau du couple : donne au lecteur une vision de
    rigueur, propreté et encaderement, le mot meme est accentué ce qui prononce l'ordre du lieu 
    - enumeration ligne 15-16 vient solidifier l'idée d'ideal, avec du vocabulaire melioratif face a la description premiere de
    l'appartement
- en contraste avec une réalité 
    - bienfaisante / insuportable : antithese : le sentiment de confusion, desordre est renforcé
    - modalisateur "sans doute" l. 1 : hypothetique futur, indique des la premiere phrase au lecteur qu'on decrit un potentiel pas la
    réalité
    - bois sale : adjectif qualificatif contraste avec le champ lexical du bois 
    - Les phrases son plus courtes : le recit est moins dans la desctiption : moins d'importance dans ce cadre
    - 2e paragraphe est introduit par une conjonction de coordination d'opposition "mais", le lecteur s'attend donc a voir la
    vision tres perfectionée de l'appartement se faire changer

# 2. une critique de PEREC de par le pathetique
- une description du couple pathetique
    - defectueuse, disgracieuse, desordre, grossiere champ lexical de la melpropreté, mauvais etat contraste avec la precendente
    description : idée de realité brutale face a imaginaire parfait
    - champ lexical de la paralisies, de la nulité bref bla bla bla
    - effreyait : mot pejoratif qui vient contraster avec les mots comme "ingenieux" "chaleureux" de manière antithetique
    - transformé, amenagé, remis a neuf : gradation descendante, ils esperent quelquechose, de moins en mois exigeant mais
    quelquechose : registre pathetique
    - personification du statu quo avec "maitres absolu" et le verbe reigner : renforce l'image mentale du lecteur de
    l'imposibilité du changement 

- critique du cout de la vie pour le couple
    - emprunter, economiser, investir : gradation + asyndète -> sentiment de peur ou d'intimidation est donné au coté financier
    - architectes, entrepreneurs, ... peintres : enumeration : avec les problemes financier le lecteur comprend que c'est
    infaisable, renforcé par l'enumeration qui appui sur le nombre d'element large  
    - negation 

- un mirroir vers une realité de son epoque 
    - le couple est addrèssé toujours ensemble dans ce paragrpahe : union commune face a la description de la realité, les deux ont
    les memes caracteristiques que les gens en general
    - realisme spacio temporel 
    - retour de la description idéal : renforce encore le pathetique
